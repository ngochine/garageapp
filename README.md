# GarageApp
Dự án phục vụ bài tập lớn môn Công nghệ phần mềm. Với đề tài được đặt ra là ***Xây dựng hệ thống quản lý trung tâm sửa xe***
...

## Hướng dẫn cài đặt
```bash
git clone <url>
./run.sh
```

## Công Nghệ Sử Dụng

| **Thành phần** | **Ứng dụng trong dự án** |
|----------------|---------------------------|
|**Bootstrap Framework**| Frontend|
| **Flask Framework** | Backend|
| **MySQL** | Lưu trữ dữ liệu sản phẩm và lịch sử phân tích |
